# EduBot
AI-Powered Query Chatbot for ICT Academy of Kerala

## About
EduBot answers student and staff queries instantly using ICTAK's own data.

## API Endpoints
- GET `/` — Health check
- GET `/health` — Health status
- POST `/chat` — Send a question, get an answer
